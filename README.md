# 📖 About

This project is a modern Full Stack Web Application developed using **React.js** for the frontend and **Django** for the backend. It provides a clean UI, secure authentication, REST APIs, and responsive design.

---

# ✨ Features

* 🔐 Login & Register
* 👤 Role-Based Authentication
* 📊 Admin Dashboard
* 📦 Product Management
* 📂 Category Management
* 🛒 Shopping Cart
* ❤️ Wishlist
* 🔍 Search & Filter
* 📱 Fully Responsive
* ⚡ Fast Performance

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/username/project.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
